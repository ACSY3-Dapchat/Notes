Spirit & Opportunity

  • Onboard computing (for autonomy)
    • Non-System RAM
      • 10 x 2BM
      • 9MB
      • 4MB
    • 128MB RAM
    • 256MB Flash
    • 20MHz RAD6000 CPU
    • VxWorks OS
    • ~75% CPU time available
  • Stereo Image Processing
    • "Combination of any camera pair & software for measuring the shape of the surrounding terrain from these images"
    • Geometric lens calibration
    • Windowed 1D search using Sum-of-Absolute-Differences
    • Generates full 3D measurements of points in stereo images
    • Opportunity
      • 48,000 x 3D Measurements
      • NAVCAMs
      • 45° FOV
    • Spirit
      • 15,000 x 3D Measurements
      • HAZCAMs
      • 125° FOV
    • Downsampled to 256x256 pixels
  • Global Path Selector
    • Field D* Planner
    • 50x50m²
    • Terrain Assessment & Local Path Selector
      • Only small obstacles
  • Local Path Selector
    • Corrects path as it drives toward some pre-specified goal location
    • Simple = no vision processing
      • next step derived from wheel encoders & gyros in IMU
    • Combine with Terrain Assessment
      • Detect & Steer around obstacle
    • Combine with Visual Odometry
      • Sandy areas
      • High slopes
  • Visual Target Tracking
    • Specify feature by appearance rather than location
    • Series of images processed to update knowledge
  • Visual Odometry
    • Accurate instrument pointing
    • Detects & tracks features
    • Bad when:
      • crossing sandy ripples
      • mixed sand/rock terrain
      • steep hillsides
    • Off by 3%
    • Relatively level ground, good position estimation
  • Autonomous Science
    • Reduce data volume
    • Detect presence of dust devils & clouds
    • Selectively transmit interesting data
  • Descent Image Motion Estimates System (DIMES)
    • 3x images taken 2.5 seconds apart
    • Estimate horizontal velocity upon landing
    • Leverages existing computer vision techniques & algorithms
    • Project 3 images onto nominal ground plane - choose & track 2 surface features
    • Radar & Attitude Measurements (Orientation)
    • Fields information to rocket motors
    • Starts at altitude of 2000m
  • Absolute Orientation Sensing
    • Onboard position estimate can accumulate drift
    • PANCAM provides an absolute heading measurement
    • Estimate by combining
      • a vector pointed at the sun
      • direction of gravity measured by IMO accelerometers
      • knowledge of current local solar time
  • Terrain Assessment
    • Processes cloud of 3D points computed by Stereo Vision
    • Fit rover-sized patches of data to a plane
    • One Spirit trip using this = 50% faster
    • Traversibility Map
      • Spirit: 10x10m²
      • Opportunity: 12x12m²
    • GESTALT (Grid Estimation of Surface Traversibility Applied to Local Terrain)
    • Geometric Hazards
      • Step Obstacles
      • Tilt Hazards
      • Roughness Hazards
  • Instrument Placement
    • Any volume not confirmed to be free of obstacles is considered unsafe
    • AutoPlace
    • Uses:
      • Trajectory generation
      • Workspace safety analysis
      • Target Selection
      • Stereo imaging
